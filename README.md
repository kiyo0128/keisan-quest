# けいさんクエスト ～ブロックワールドの冒険～

引き算を楽しく学べる子供向け計算アドベンチャーゲーム 🎮

マインクラフト風のブロックワールドで冒険しながら、引き算をマスターしよう！

## 🎯 特徴

- **⚔️ バトルシステム** — モンスターと引き算バトル！正解でダメージ、間違えるとピンチ
- **⛏️ 採掘（さいくつ）** — ブロックを掘って素材をゲット！引き算で掘り進もう
- **� クラフト** — 集めた素材で武器・防具を作ってパワーアップ
- **📈 自動難易度調整** — お子様の正答率に合わせて問題の難しさが変化
- **🔥 コンボシステム** — 連続正解でボーナスダメージ＆スコアアップ
- **💾 セーブ機能** — ブラウザに自動保存、いつでも続きから遊べる

## 🕹️ 遊び方

1. ブラウザで `index.html` を開く
2. 「はじめる」でスタート
3. ハブ画面から **バトル・採掘・クラフト** を選んで冒険！

### 計算の難易度

| レベル | 範囲 | 例 |
|--------|-------|------|
| かんたん 🌱 | 10以下 | 8 - 3 = ? |
| ふつう ⚡ | 11～18 | 16 - 9 = ? |
| むずかしい 🔥 | 12～19（繰り下がり） | 15 - 8 = ? |

## 🛠️ 技術スタック

- HTML / CSS / JavaScript（バニラ、フレームワーク不使用）
- ピクセルアートテーマ（DotGothic16 + Press Start 2P フォント）
- 8bit効果音（Web Audio API）
- LocalStorage によるデータ保存
